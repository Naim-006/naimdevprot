# OS Portfolio — Developer Workspace

Interactive OS-themed developer portfolio featuring Windows 11 desktop and iOS mobile experiences, with full CRUD admin control and multilingual support.

## Features

- **Windows 11 Desktop** — Draggable windows, taskbar, start menu, system tray, snap layouts
- **iOS Mobile** — Home screen widgets, app grid, dock, swipe gestures, app switcher
- **10 Portfolio Apps** — About, Skills, Projects, Experience, Testimonials, Contact, Blog, Terminal, Settings, Admin
- **Admin Panel** — Password-protected full CRUD dashboard for all portfolio data
- **Multilingual** — English, Bangla, Arabic, Chinese, German, Russian
- **Lock Screen** — Click/swipe to unlock
- **Notification Center** — Integrated with calendar flyout
- **Spotlight Search** — Ctrl+K to search apps, projects, skills
- **Keyboard Shortcuts** — Alt+F4, Ctrl+D, Esc, Delete
- **Live Battery** — Real battery percentage via Battery Status API

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```
